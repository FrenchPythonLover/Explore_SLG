# Saint-Louis de Gonzague Perpignan - 3D Recreation
Projet open-source de reproduction 3D réaliste et explorable de l'établissement.  
Fait avec le Godot Engine.  
Version web et windows disponible prochainement.  

## Comment voir/modifier/compiler le code  
Pour voir, modifier ou compiler le code et les objets 3d, une manière est d'ouvrir ce repo sous la forme d'un projet Godot (Godot est un moteur de jeu).  
Téléchargez Godot **3** (3, pas 4): [Windows](https://downloads.godotengine.org/?version=3.6.2&flavor=stable&slug=win64.exe.zip&platform=windows.64) [Linux](https://downloads.godotengine.org/?version=3.6.2&flavor=stable&slug=x11.64.zip&platform=linux.64) [macOS](https://downloads.godotengine.org/?version=3.6.2&flavor=stable&slug=osx.universal.zip&platform=macos.universal)  
Puis, téléchargez le zip de ce projet [ici](https://github.com/FrenchPythonLover/Explore_SLG/archive/refs/heads/main.zip), extrayez le dossier game (il contient la source, le reste c'et la version web). puis ouvrez le depuis godot.
De là, vous pouvez explorer le projet librement (La première scene exécutée est dans scenes/menu/principal.tscn).
